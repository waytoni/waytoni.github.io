අනිච‍්චං --- අනිස ය (අනිසි-නිසි නැති): අනාගත යදනිච්ච සූත්‍රය සහ පච්චුප්පන්න යදනිච්ච සූත්‍රය

සංයුත්ත නිකාය → ඛන්ධක වග්ගෝ → හේතු සූත්‍රය

සංයුත්ත නිකාය → ඛන්ධක වග්ගෝ → පඨම අස්සාද සුත්තං

අංගුත්තර නිකාය → ඒකාදසක නිපාතතෝ → පඨම සඤ්ඤා සුත්තං

සංයුත්ත නිකාය → ඛන්ධක වග්ගෝ → දුතිය අස්සාද සුත්තං

සංයුත්ත නිකාය → නිදාන වර්ගය → අභිසමය සංයුත්තං → මහා වර්ගය → සුසීම සුත්තං

මජ්ඣිම නිකාය → මූල පණ්ණාසකකෝ → සබ්බාසව සුත්තං

සංයුත්ත නිකාය → සළායතන සංයුත්තය → සබ්බ වර්‍ගය → සබ්බ සූත්‍රය සහ ප්‍රහාණ සූත්‍රය

සංයුත්ත නිකාය → නිදාන වර්‍ගය → අභිසමය සංයුත්තය → බුද්ධ වර්‍ගය → විභඞ්ග සූත්‍රය
